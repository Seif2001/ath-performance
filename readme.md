## 🛠 Tech Stack

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- ORM: Prisma
- Database: PostgreSQL
- Dev Tools: Docker

## 🚀 How to Run

```bash
#simply run:
git clone github.com/Seif2001/ath-performance.git
cd ath-performance
docker compose up
```
